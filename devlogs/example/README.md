# Dev Log Instructions

## How to create a new dev log?

In `/devlogs/your-name/`, create a new `devlog-n.md` where `n` is the dev log number.

## When to create a new dev log?

- Create a dev log whenever you make meaningful progress on the project.
- A dev log may describe work on a feature, debugging, testing, or research.
- Write the dev log at the end of your work session.
- A dev log can/should be created even if you didn't accomplish your goal(s).
