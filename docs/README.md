# Documentation

## What will this folder include?

- Circuit diagrams.
- UML diagrams.
- Demo photos & videos.
- Similar non-code documents.

## How should I name my file?

- It is upto you, but it should be descriptive e.g. if it's a circuit diagram, mention that.
- **(Optional)** If the file is iterative (you're changing it constantly), version numbers can be useful.
